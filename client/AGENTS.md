# AGENTS.md

## 1. Project overview

- **Name:** Finance X-Ray
- **What it does:** The application is used to show analytics (graphics) on top of uploaded csv files, containing data for movement of transactions of the users personal bank account.
- **Status:** Prototype
- **Primary users:** For personal usage

## 2. Tech stack

- React 19 + TypeScript
- Vite (build tool / dev server)
- MUI (Material-UI) for components
